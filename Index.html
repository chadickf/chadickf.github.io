TAREA SOBRE HTML

Lista de Elementos HTML Comunes

  • Encabezados: <h1> a <h6>
  • Párrafos: <p>
  • Enlaces: <a>
  • Imágenes: <img>
  • Listas: <ul>, <ol>, <li>
  • Divisiones: <div>
  • Span: <span>
  • Tablas: <table>, <tr>, <td>, <th>
  • Formularios: <form>, <input>, <button>

Tabla de Atributos HTML Comunes

Atributo Descripción
class Define una o más clases para un elemento.
id Define un identificador único para un elemento.
src Especifica la URL de un recurso externo (imágenes, videos, etc.).
href Especifica la URL de un enlace.
alt Proporciona texto alternativo para una imagen.
